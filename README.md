# 🚀 DSA in C and C++

A comprehensive collection of **Data Structures and Algorithms (DSA)** implemented in **C** and **C++**. This repository is designed for students, beginners, and coding interview aspirants who want to build a strong foundation in problem-solving and programming.

## 📌 Objectives

- Learn Data Structures and Algorithms from scratch
- Strengthen problem-solving skills
- Practice coding interview questions
- Improve coding logic using C and C++
- Prepare for technical interviews and competitive programming

## 📚 Topics Covered

- Basics of C & C++
- Arrays
- Strings
- Functions
- Pointers
- Recursion
- Sorting Algorithms
- Searching Algorithms
- Linked List
- Stack
- Queue
- Trees
- Binary Search Tree
- Heap
- Graph
- Hashing
- Dynamic Programming
- Greedy Algorithms
- Backtracking
- Bit Manipulation
